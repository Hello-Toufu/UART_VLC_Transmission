RESET_CTRL Offset = 16'h00 Attributes : RW
soft reset
Field Name Bits Default Description
soft_all_rst 0:0 1'b0 reset all modules except cfg
soft_cfg_rst 1:1 1'b0 reset top cfg module


RW_TEST Offset = 16'hFF Attributes : RW
use to test the register access
Field Name Bits Default Description
rw_scratch 15:0 16'h0 scratch board for test


PGEN_CTRL1 Offset = 16'h10 Attributes : RW
packet generator control
Field Name Bits Default Description
tx_gen_en 0:0 1'b0 1' to enable packet generator
tx_num_clr 4:4 1'b0 1' to clear the packet counter


PGEN_ERR_CTRL Offset = 16'h11 Attributes : RW
insert prbs error to packet source
Field Name Bits Default Description
tx_txperrone 0:0 1'b0 change from '0' to '1' will insert 1 error


PGEN_PKT_CNT Offset = 16'h12 Attributes : RO
packet generator packet num
Field Name Bits Default Description
tx_pkt_num 15:0 16'h0 sent packet num


PGEN_CTRL2 Offset = 16'h13 Attributes : RW
pkt interval
Field Name Bits Default Description
tx_pkt_interval 15:0 16'd800 packet interval


PCHK_CTRL Offset = 16'h14 Attributes : RW
packet checker control
Field Name Bits Default Description
rx_chk_en 0:0 1'b0 1' to enable packet checker
rx_num_clr 4:4 1'b0 1' to clear the packet counter


PCHK_ERR_CNT Offset = 16'h15 Attributes : RO
packet prbs error counter
Field Name Bits Default Description
rx_errcnt 15:0 16'h0 prbs error counter


PCHK_PKT_CNT Offset = 16'h16 Attributes : RO
packet checker packet num
Field Name Bits Default Description
rx_pkt_num 15:0 16'h0 receive packet num


ERR_INJ_CTRL Offset = 16'h20 Attributes : RW
line error injection control
Field Name Bits Default Description
err_inj_cont 0:0 1'b0 continues insert error


SIM_CABLE_DIS Offset = 16'h21 Attributes : RW
simulate cable disconnection
Field Name Bits Default Description
lane_cable_dis 3:0 4'h0 1' will disconnect the serial data (set to 0)


S2P_PHASE Offset = 16'h30 Attributes : RW
select use posedge or negedge to sample serial data in s2p
Field Name Bits Default Description
s2p_phase 3:0 4'h0 1' will use negedge, '0' will use posedge


DBG_OUTPORT Offset = 16'h80 Attributes : RW
debug signals
Field Name Bits Default Description
debug_out1 15:0 16'h0 debug signal


DBG_INPORT Offset = 16'h81 Attributes : RO
debug signals
Field Name Bits Default Description
debug_in1 15:0 16'h0 debug signal


PGEN_CTRL3 Offset = 16'h17 Attributes : RW
packet length
Field Name Bits Default Description
tx_pkt_len 13:0 16'd1000 packet length of byte

valid range: 44-1498
tx_pkt_fix_len 15:15 1'b1 0: packet length increase from 64 to 1514

1: packet length is set by tx_pkt_len
